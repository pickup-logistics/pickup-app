# PickUp API - Postman/Thunderclient Collections

## Import Instructions

1. Open Postman
2. Click Import
3. Select the .json file
4. Set environment variables:
   - `baseUrl`: http://localhost:5001/api/v1

## Collections

- **PickUp_Auth.json** - Authentication endpoints
- **PickUp_Rides.json** - Ride booking (Phase 5)
- **PickUp_Admin.json** - Admin endpoints (Phase 6)
